PROJECTS

Advanced Poultry Management:

An automated sensor system to monitor ammonia/H2S emissions in poultry litter, featuring Ammonia Gas Sensors, Steamer Technology for neutralization, Exhaust Fans for odor control, and Microcontroller Integration for seamless operation. Additional sensors for filtration, fly deterrence, and temperature optimize efficiency, perfect for modern poultry farming.

Automated Stain Detection

Our machine learning-based system, utilizing OpenCV and Tkinter, detects dirt on clothes, enhancing textile industry quality control. It alerts workers to impurities, boosting efficiency, product quality, and brand reputation, driving productivity and upholding high standards in textile manufacturing through innovative technology integration.

Real-Time Earth Leakage Detection

Real-time Earth Leakage Detection system for parking lights and electric poles using CT sensors. LoRa-based communication ensures efficient long-range data transmission, with instant alerts and location mapping via an app. Arduino-based setup supports offline functionality for remote areas, requiring internet only for the LoRa WAN gateway.
